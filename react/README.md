#WX-route-planer
基于微信端的旅游拼团h5应用，当前版本提供简单的拼团流程

#Features
+ 浏览旅游拼团信息
+ 参团、微信支付功能
+ 微信分享功能

#Notice
+ 常规分支：
  + master         线上分支
  + develop        主开发分支
+ 临时分支：
  + hot fix        线上bug紧急修复分支（上线时间戳为标识）
  + release        日常分支的提前开发分支（上线时间戳为标识）
  + feature        新版本开发分支（平台/版本号）
  + tmp            临时合并测试分支（测试时间戳
